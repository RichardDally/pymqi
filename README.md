# Pymqi docker image
Ready to use Docker container for IBM MQ [pymqi](https://pypi.org/project/pymqi) technology.

### Sources

Based on a home made [CPython docker image](https://github.com/RichardDally/CPython). 

### Current version (i.e. latest tag)

- CPython 3.8.2
- Ubuntu 18.04

### Docker Hub

[DockerHub](https://hub.docker.com/r/richarddally/pymqi) is configured to build and publish containers.
